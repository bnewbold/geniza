Paper says:

    Ed25519 sign(
        BLAKE2b(
            <1 byte> 2 // root type
            for (every root node left-to-right) {
                <32 byte root hash>
                <8 byte Uint64BE root tree index>
                <8 byte Uint64BE child byte lengths>
            }
        )
    )
